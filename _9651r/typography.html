Overview

At 9651 Robotics, we like robots, and anything related to them. That's why we use Google's Roboto and Roboto Slab as our main font families for all of our material: not only does it look and feel cool, but it has to do with robots. Awesome!

Roboto

Roboto is the normal sans-serif font family for 9651 Robotics media materials. Use it for general text, html <p> tags, and any other default text values. When in doubt, use Roboto.

Roboto Slab

Roboto Slab is the default serif font family for 9651 Robotics media materials. Use it for headings, html header tags, and for any title or heading-based text.

Font Sizing

Defaults

We recommend a default font-size of at least 16px for digital media, and at least 14pt for print media. This is to ensure legibility, especially from a distance. 

    
      $font-size-root: 16px !default;

Headers

We recommend a header size of at least 1.25 em (20px or 16pt). Here's an example set of headers that we use in our Bootstrap theme. 

    
      $font-size-root: 16px !default;

      $font-size-h1: 2.5rem !default;
      $font-size-h2: 2rem !default;
      $font-size-h3: 1.75rem !default;
      $font-size-h4: 1.5rem !default;
      $font-size-h5: 1.25rem !default;
      $font-size-h6: 1rem !default;

Line Height

We recommend a line-height of at least 1.5, to ensure legibility and spacing.

CSS Example

Here's some example code using the Roboto and Roboto Slab core typefaces.

SASS Example

    
      // Font Imports

      @font-face{
      	font-family: 'Roboto';
      	src:url({{site.baseurl}}/css/font/roboto/Roboto-Regular.ttf);
      }

      @font-face{
      	font-family: 'Roboto Slab';
      	src:url({{site.baseurl}}/css/font/robotoslab/RobotoSlab-Regular.ttf);
      }

      // Font Classes

      .roboto{
      	font-family: 'Roboto';
      }

      .roboto-slab{
      	font-family: 'Roboto Slab';
      }
