Overview

Typography is a key element of any graphical or text-based design, and the World Affairs Conference is no exception. WAC uses the Nexa font face, with several variations depending on material, type, and location. Here's an overview of how to use the WAC typography.

Basic Typeface

Aa

Nexa Bold

Nexa Bold is the main header font for the World Affairs Conference. Use it for titles, html header tags, and anything that you want an emphasis on.

Download Nexa Bold
Aa

Nexa Light

Nexa Light is the normal typeface for World Affairs Conference media materials. Use it for general text, paragraph text, and anything that's informational text.

Download Nexa Light
