Overview

Just like branding, icons are used to convey meaning and association. At the World Affairs Conference, we use two types of icons: in-house designs, and the font icon set Font Awesome. We'll go over how to use both of them!

WAC Icons

WAC Icons are coming soon! In the meantime, use Font Awesome.

Font Awesome

Font Awesome is a free and open-source icon font. It has over 600 icons, and is usable in websites, print material, and almost anyting else! WAC uses it in the WAC website, WAC print materials, signage, and other materials.

Using Font Awesome is also super easy. For general usage, you can install the icon font or use the SVGs provided on their website. For web-related usage, their website has a great walkthrough.
