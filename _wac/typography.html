Overview

Typography is a key element of any graphical or text-based design, and the World Affairs Conference is no exception. WAC uses the Nexa and Nexa Slab font faces, with several variations depending on material, type, and location. Here's a basic overview:

Font Sizing

Defaults

We recommend a default font-size of at least 16px for digital media, and at least 14pt for print media. This is to ensure legibility, especially from a distance. 

    
      $font-size-root: 16px !default;

Headers

We recommend a header size of at least 1.25 em (20px or 16pt). Here's an example set of headers that we use in our Bootstrap theme. 

    
      $font-size-root: 16px !default;

      $font-size-h1: 2.5rem !default;
      $font-size-h2: 2rem !default;
      $font-size-h3: 1.75rem !default;
      $font-size-h4: 1.5rem !default;
      $font-size-h5: 1.25rem !default;
      $font-size-h6: 1rem !default;

Line Height

We recommend a line-height of at least 1.5, to ensure legibility and spacing.

CSS Example

Here's some example code using the Nexa and Nexa Slab core typefaces.

SASS Example

    
      $text-color: #383938;

      // Font Imports

      @font-face{
      	font-family:Nexa;
      	src:url({{site.baseurl}}/css/fonts/nexa/nexa_light.otf);
      }
      @font-face{
      	font-family:Nexa;
      	font-weight:bold;
      	src:url({{site.baseurl}}/css/fonts/nexa/nexa_bold.otf);
      }
      @font-face{
      	font-family:Nexa Slab;
      	src:url({{site.baseurl}}/css/fonts/nexaslab/nexa_slab_regular.otf);
      }
      @font-face{
      	font-family:Nexa Slab;
      	font-weight: bold;
      	src:url({{site.baseurl}}/css/fonts/nexaslab/nexa_slab_bold.otf);
      }

      // Font Classes

      .nexa-light{
      	font-family: Nexa;
      }

      .nexa-bold{
        font-family: Nexa;
        font-weight: bold;
      }

      .nexa-slab{
      	font-family:Nexa Slab;
      }

      .nexa-slab-bold{
      	font-family:Nexa Slab;
      	font-weight: bold;
      }

      // Sample Usage

      .wrapper{
        font-family:Nexa;
        color: $text-color;

      	h1,h2,h3,h4,h5,h6{
      		font-family:Nexa Slab;
      	}
      }
