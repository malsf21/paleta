Overview

Typography is a key element of any graphical or text-based design, and the World Affairs Conference is no exception. WAC uses the Nexa and Nexa Slab font faces, with several variations depending on material, type, and location. Here's a basic overview:

Core Typefaces

Aa

Nexa Light

Nexa Light is the normal sans-serif typeface for World Affairs Conference media materials. Use it for general text, html <p> tags, and any other default text values. When in doubt, use Nexa Light.

Nexa Bold

Nexa Bold is the boldened or emphasized sans-serif typeface for World Affairs Conference media materials. Use it for bolded words, html font-weight:bold;, and anything that you want an emphasis on.

CSS Example

Here's some example code using the Nexa and Nexa Slab core typefaces.

SASS Example

    
      $text-color: #383938;

      // Font Imports

      @font-face{
      	font-family:Nexa;
      	src:url({{site.baseurl}}/fonts/nexa/nexa_light.otf);
      }
      @font-face{
      	font-family:Nexa;
      	font-weight:bold;
      	src:url({{site.baseurl}}/fonts/nexa/nexa_bold.otf);
      }
      @font-face{
      	font-family:Nexa Slab;
      	src:url({{site.baseurl}}/fonts/nexaslab/nexa_slab_regular.otf);
      }
      @font-face{
      	font-family:Nexa Slab;
      	font-weight: bold;
      	src:url({{site.baseurl}}/fonts/nexaslab/nexa_slab_bold.otf);
      }

      // Font Classes

      .nexa-light{
      	font-family: Nexa;
      }

      .nexa-bold{
        font-family: Nexa;
        font-weight: bold;
      }

      .nexa-slab{
      	font-family:Nexa Slab;
      }

      .nexa-slab-bold{
      	font-family:Nexa Slab;
      	font-weight: bold;
      }

      // Sample Usage

      .wrapper{
        font-family:Nexa;
        color: $text-color;

      	h1,h2,h3,h4,h5,h6{
      		font-family:Nexa Slab;
      	}
      }
