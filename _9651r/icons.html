Overview

Just like branding, icons are used to convey meaning and association. At 9651 Robotics, we use the font icon set Font Awesome. Let's talk about just how awesome it is!

Font Awesome

Font Awesome is a free and open-source icon font. It has over 600 icons, and is usable in websites, print material, and almost anyting else! 9651 uses it in the 9651 website, 9651 print materials, signage, and other materials.

Using Font Awesome is also super easy. For general usage, you can install the icon font or use the SVGs provided on their website. For web-related usage, their website has a great walkthrough.
